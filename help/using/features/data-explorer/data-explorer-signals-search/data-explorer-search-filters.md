# 按筛选条件搜索信号 {#search-signals-by-filters}

将键值字段留空以搜索范围更广的信号，并使用可用的过滤器来缩小结果范围。

当您没有考虑特定的键值对，但希望了解特定时间段内多个信号的演变情况时，请使用此方法。

在以下示例中，过滤器配置为显示过去30天中所有未使用的信号，最少计数为1000。
