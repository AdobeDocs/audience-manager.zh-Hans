# Search Signals by Filters {#search-signals-by-filters}

将key-value字段保留为空可搜索更广泛的信号，并使用可用的过滤器缩小结果。

当您心中没有特定的键值对时，请使用此方法，但要在特定时间段内查看多个信号的演变。

在以下示例中，过滤器配置为在过去30天内显示所有未使用的信号，最低计数为1000。
