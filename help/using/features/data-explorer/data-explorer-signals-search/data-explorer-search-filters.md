# 按筛选条件搜索信号 {#search-signals-by-filters}

将键值字段留空可搜索范围更广的信号，并使用可用的过滤器缩小结果范围。

当您没有考虑特定的键值对，但希望查看多个信号在特定时间段内的演变时，请使用此方法。

在下面的示例中，过滤器配置为显示过去30天中的所有未使用信号，最小计数为1000。
