# 区段列表视图 {#segments-list-view}

功能 [!UICONTROL Segments] 板是用于管理目标的集中式工作区。

主页包 [!UICONTROL Segments] 含帮助您：

* 创建新细分。
* 编辑和删除区段。
* 克隆（复制）现有区段。
* 通过可排序的列查看表中的所有区段。
* 管理细分存储。
* 按名称搜索区段。
