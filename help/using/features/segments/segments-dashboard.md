# Segments List View {#segments-list-view}

[!UICONTROL Segments] 仪表板是用于管理目标的中央工作区。

The main [!UICONTROL Segments] page contains features and tools that help you:

* 创建新细分。
* 编辑和删除区段。
* 克隆(复制)现有区段。
* 在表格中查看带有可排序列的所有区段。
* 管理细分存储。
* 按名称搜索区段。
