# 交互式报表和重叠报表{#interactive-and-overlap-reports}

交互式报表显示特征和区段的性能和重叠数据。 这些报表不使用列和行中排列的数字，而是使用不同的形状、颜色和大小返回数据。 此外，您还可以选择单个或一组数据点，并深入到报表结果以了解更多详细信息。 这些可视化技术和报表交互性有助于使大量数字数据更易于理解。
