# 使用数据滑块筛选报表结果{#filter-report-results-with-the-data-sliders}

使用各种报告滑块仅显示位于指定范围、上方或下方的数据。

<!--

 -->

**使用数据滑块设置报告结果的下限／上限范围**

通过报表滑块，您可以设置对交互式报表返回的数据的限制。 移动左滑块，排除特定值以下的数据。 移动右滑块以排除特定值以上的数据。 报表会更新并返回所需范围内的数据。 使用滑块可以：

* 减少报告返回的总数据量。
* 关注特定大小范围内的特征或细分。
