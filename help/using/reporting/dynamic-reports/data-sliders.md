# 使用数据滑块筛选报表结果{#filter-report-results-with-the-data-sliders}

使用各种报表滑块仅显示位于指定范围内、上方或下方的数据。

<!--

 -->

**使用数据滑块设置报表结果的下限/上限范围**

通过报表滑块，您可以设置对交互式报表返回的数据的限制。 向左移动滑块可排除低于特定值的数据。 移动右滑块可排除特定值以上的数据。 报表更新并返回所需范围内的数据。 使用滑块可以：

* 减少报告返回的总体数据量。
* 关注特定大小范围内的特征或细分。
