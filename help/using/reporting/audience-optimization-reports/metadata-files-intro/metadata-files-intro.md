# 受众优化报告的数据和元数据文件{#data-and-metadata-files-for-audience-optimization-reports}

数据文件包含可导入“受众优化”报告的印象、单击或转化数据。 元数据文件包含与各种报告选项和菜单项相对应的可读名称。 根据本节中的规范设置数据和元数据文件的格式。

>
