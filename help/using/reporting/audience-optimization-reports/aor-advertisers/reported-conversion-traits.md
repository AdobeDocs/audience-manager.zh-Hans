# 报告的转化特征{#reported-conversion-traits}

“转化特征”报表显示在特定日期标记为转化群组转化特征的所有特征。

转化组的转化特征可能会从报表运行更改为报表运行。 报表按转化群组显示选定报表日期的转化特征。
