# 报告的转化特征{#reported-conversion-traits}

“转化特征”报告显示在某个日期标记为转化组的转换特征的所有特征。

转换组的转换特征可以从报告运行更改为报告运行。 该报表按选定报告日期的转化组显示转化特征。
