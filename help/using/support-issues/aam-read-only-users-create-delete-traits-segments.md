# 为什么具有只读权限的用户能够创建、编辑或删除特征和区段？

## 问题

为什么具有只读权限的用户能够创建、编辑或删除特征和区段？

## 回答

除了在管理部分中创建组和权限之外，您还需要联系客户关怀团队 (`amsupport@adobe.com`)，请求代表为您的帐户启用[基于角色的访问控制 (RBAC)](../features/administration/administration-overview.md)。
