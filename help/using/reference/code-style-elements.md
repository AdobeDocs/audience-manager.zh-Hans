# Style Conventions for Code and Text Elements{#style-conventions-for-code-and-text-elements}

这些元素识别了整个帮助文档中使用的代码选项和变量。通常，您不会在代码或数据文件中包含这些符号或样式元素。它们只是可视指示器。

  </tr>
